# Priklad na statnice:
-Dokazte ze nasobeni matic neni komutativni
- Bod A[1, 1, 1], Rotace(90, x), Translace(0, 5, 0)

  - Nejprve (priklad 1) spocitam rotaci a pak translaci a nasledne (priklad 2) translaci a rotaci. Pokud to vyjde stejne, tak jsou komutativni

  - priklad 1
    - prevedu si bod na homogenni souradnice [1 1 1 1]
    - zapisu si rotaci
      [ 1     0      0     0 ]
      [ 0   cos90  -sin90  0 ]
      [ 0   sin90  -cos90  0 ]
      [ 0     0      0     1 ]
    - Zmenim cos/sin na cisla
    [ 1 0  0 0 ]
    [ 0 0 -1   ]
